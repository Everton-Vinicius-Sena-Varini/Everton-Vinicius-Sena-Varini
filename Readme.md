# Sobre o projeto

Este projeto está sendo desenvolvido como abordagem de aprendizado baseado em projetos e como parte do meu repositório como estudante de tecnologia na
área de Desenvolvimento Web. A ideia é criar uma Web Site interativo para introduzir iniciantes no jogo Brawlhalla as mecânicas do jogo e aos jogos de luta
desse estilo, tendo como objetivo de aprendizado e desenvolvimento de competências, obter a capacidade de criar Web Sites com UX/UI Design consistentes
um bom Web e UX Writing organizando as informações em uma interface moderna, componentizada, organizada, acessível e segura.
